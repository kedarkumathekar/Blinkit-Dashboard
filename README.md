# Blinkit Sales Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes Blinkit's sales and operational performance using various business metrics and interactive visualizations. The objective of this project is to transform raw retail data into actionable insights that help understand sales performance, customer behavior, outlet efficiency, and product trends.

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Sales
* Number of Items Sold
* Average Sales
* Average Customer Rating

### Analysis Performed

* Sales by Fat Content
* Sales by Item Type
* Outlet Establishment Trend
* Sales by Outlet Size
* Sales by Outlet Location
* Outlet Type Performance Analysis
* Customer Rating Analysis

## Business Questions Answered

* Which product categories generate the highest sales?
* How does outlet size impact revenue?
* Which outlet locations perform best?
* What are the sales trends across different outlet types?
* How do customer ratings vary across products and outlets?

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Visualization
* Data Cleaning & Transformation

## Dashboard Insights

* Identified top-performing product categories.
* Analyzed revenue contribution across outlet sizes and locations.
* Evaluated customer satisfaction through rating analysis.
* Compared sales performance among different outlet types.
* Tracked overall business performance using KPI cards.
